# News Tabs Toggle Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Секция новостей получает две плашки-таба («Jaunumi» / «Noderīgi raksti»), переключающие два Swiper-слайдера; виден всегда один.

**Architecture:** Два слайдера в DOM (show/hide через `hidden`). Конфиг Swiper вынесен в фабрику `createNewsSwiper(selector)` с идемпотентной инициализацией (WeakMap по DOM-элементу). Новый модуль `newsTabs.js` переключает табы, лениво инициализирует скрытый слайдер при первом показе и управляет autoplay.

**Tech Stack:** Gulp 4 (сборка), Webpack (JS-бандл), Swiper 11, SCSS, gulp-file-include (HTML-блоки), ScrollReveal.

**Spec:** `docs/superpowers/specs/2026-07-08-news-tabs-design.md`

## Global Constraints

- Язык контента: латышский.
- Заголовок секции: «Jaunumi un informācija»; табы: «Jaunumi», «Noderīgi raksti».
- Стиль плашек: без скруглений, uppercase, `font-weight: 700`; активная — фон `var(--accent)` + белый текст, неактивная — фон `var(--accent-tint)` + `var(--text-color)`.

### Task 3: JS — фабрика Swiper

**Files:**
- Modify: `src/js/modules/swipers.js` (полная замена содержимого)

**Interfaces:**
- Consumes: DOM из Task 1 (`.swiper-news`).
- Produces: именованный экспорт `createNewsSwiper(selector: string): Swiper | null` — идемпотентная фабрика (повторный вызов для того же DOM-элемента возвращает существующий инстанс, даже при другом селекторе). Дефолтный экспорт `initAllSwipers(): void` — как раньше, инициализирует видимый `.swiper-news`.

- [ ] **Step 1: Заменить содержимое `src/js/modules/swipers.js` целиком**

```js
import Swiper from "swiper";
import { Autoplay } from "swiper/modules";
import "swiper/css";

const newsSwiperConfig = {
  modules: [Autoplay],
  loop: false,
  slidesPerView: 3,
  spaceBetween: 20,

  breakpoints: {
    0: { slidesPerView: 1 },
    760: { slidesPerView: 2 },
    1024: { slidesPerView: 3 },
  },

  autoplay: {
    delay: 2500,
    disableOnInteraction: false,
    reverseDirection: false,
    pauseOnMouseEnter: true,
  },
  speed: 700,
};

const instances = new WeakMap();

export function createNewsSwiper(selector) {
  const container = document.querySelector(selector);
  if (!container) return null;
  if (instances.has(container)) return instances.get(container);

  const swiper = new Swiper(container, newsSwiperConfig);
  instances.set(container, swiper);
  return swiper;
}

export default function initAllSwipers() {
  createNewsSwiper(".swiper-news");
}
```

Примечание: WeakMap по элементу (не по селектору) — `initAllSwipers` обращается через `.swiper-news`, а `newsTabs` (Task 4) через `#news-panel-news`; это один элемент, инстанс один.

**Interfaces:**

- Produces: дефолтный экспорт `newsTabs(): void`, вызывается один раз при загрузке.

- [ ] **Step 1: Создать `src/js/modules/newsTabs.js`**

```js
import { createNewsSwiper } from "./swipers.js";

export default function newsTabs() {
  const tablist = document.querySelector(".news__tabs");
  if (!tablist) return;

  const tabs = Array.from(tablist.querySelectorAll(".news__tab"));

  tabs.forEach((tab) => {
    tab.addEventListener("click", () => {
      if (tab.classList.contains("news__tab--active")) return;

      tabs.forEach((current) => {
        const isActive = current === tab;
        current.classList.toggle("news__tab--active", isActive);
        current.setAttribute("aria-selected", String(isActive));

        const panel = document.getElementById(
          current.getAttribute("aria-controls"),
        );
        if (!panel) return;

        // hidden снимается до создания Swiper — иначе ширина слайдов = 0
        panel.hidden = !isActive;

        const swiper = createNewsSwiper(`#${panel.id}`);
        if (!swiper) return;

        if (isActive) {
          swiper.autoplay.start();
        } else {
          swiper.autoplay.stop();
        }
      });
    });
  });
}
```

- [ ] **Step 3: Сузить селектор ScrollReveal**

В `src/js/modules/scrollReveal.js` строка 59, было:

```js
  ScrollReveal().reveal(`.news__slide, .products__card, .btn-link`, {
```

стало:

```js
  ScrollReveal().reveal(`.swiper-news .news__slide, .products__card, .btn-link`, {
```

Причина: ScrollReveal ставит слайдам `opacity: 0` до появления в вьюпорте; слайды внутри `hidden`-панели в вьюпорт не попадают и после переключения таба остались бы невидимыми. Reveal-анимация остаётся только у изначально видимого слайдера.
